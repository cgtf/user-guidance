      • 2要素認証を利用する
    • NFT
      • 販売元にNFTを販売する権利がなかった

      • NFTの販売情報が偽物だった
      • NFTの価格が大きく変動するリスク
      • 売りたい時に買い手が現れないリスク
      • ウォッシュトレードによる見せかけの価格
      • NFTを先行販売したプロジェクトにその後の進展がない
    • スマートコントラクト・DeFi
      • プログラムの不具合による資金流出
